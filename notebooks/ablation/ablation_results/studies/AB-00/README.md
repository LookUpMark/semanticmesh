# AB-00 Artifacts

This folder contains all generated artifacts for study AB-00.

## Subfolders

- `runs/`: run output and execution logs.
- `traces/`: query-level and RAGAS traces.
- `diagnostics/`: structured diagnostics.
- `analyses/`: manual and iterative sample analyses.
- `reports/`: reliability review and audit documents.
- `baselines/`: baseline outputs used for A/B comparison.
